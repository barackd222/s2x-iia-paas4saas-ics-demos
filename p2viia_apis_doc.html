Conference Call

/confcall/{fromNumber}/{toNumber}/{recipientName} post

post

Establishes a conference call across 2 people. The first phone number in the URI represents the caller, the second the recipient.

post /confcall/{fromNumber}/{toNumber}/{recipientName}

  • Request
  • Response

URI Parameters

  • fromNumber: required (string)
  • toNumber: required (string)
  • recipientName: required (string)

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
  "message": "Thank you! We will be calling you shortly."
}

Social Channel Manager

/social/message post

post

Pulls marketing campaigns out of Salesforce and posts messages into Social Media (LinkedIn and Twitter) including the selected campaigns.

post /social/message

  • Response
